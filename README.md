# Özad Elektrotechniek B.V.

Modern website redesign for Özad Elektrotechniek B.V. — an electrician company based in Rotterdam.

## Live Site

[ozadelektrotechniek.nl](https://ozadelektrotechniek.nl)

## Built With

- Next.js
- Tailwind CSS
- TypeScript
- shadcn/ui
- Framer Motion
- Resend

## Features

- Fully responsive design
- Framer Motion animations
- Contact form with spam protection
- SEO optimized
- 90+ PageSpeed score on mobile and desktop
